# Pro & Team IP Website

## Setup & Run

```bash
# 1. Install dependencies
npm install

# 2. Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Pages
- `/` - Home
- `/team` - Our Team
- `/services` - Services
- `/fee-structure` - Fee Structure
- `/contact` - Contact

## Build for production
```bash
npm run build
npm run start
```
